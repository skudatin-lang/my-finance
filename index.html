2. Settings → Variables → добавить:
  • YANDEX_API_KEY = ключ с console.yandex.cloud
  • YANDEX_FOLDER = folder ID
  • APP_SECRET = любой пароль
  • ANTHROPIC_KEY = ключ с console.anthropic.com (для ИИ)
3. Deploy → скопировать URL → вставить выше
4. Нажать «Проверить» — должно быть ✓
