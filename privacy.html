Политика конфиденциальности

Редакция от 18 апреля 2026 г.
Настоящая Политика конфиденциальности определяет порядок обработки персональных данных пользователей сервиса «Мои Финансы» (далее — Сервис) в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

1. Оператор персональных данных

Оператором персональных данных является физическое лицо — разработчик Сервиса (самозанятый / ИП, реквизиты указываются при коммерческом запуске). Контактный адрес для запросов по персональным данным: skudatin@gmail.com

2. Какие данные мы собираем

ДанныеИсточникЦель
Email-адрес, имя, фото профиля Google OAuth при авторизации Идентификация пользователя, отображение аватара
Уникальный идентификатор (UID) Google Firebase Authentication Привязка данных к аккаунту
Финансовые данные (операции, балансы, категории) Вводятся пользователем вручную или голосом Основная функциональность Сервиса
Резервная копия данных Автоматически в localStorage браузера Восстановление при сбоях сети

Мы не собираем: данные банковских карт, номера телефонов, адреса, данные о местоположении, биометрические данные.

3. Как и где хранятся данные

Финансовые данные хранятся в облачной базе данных Google Cloud Firestore (регион eur3, Европейский союз). Данные передаются по зашифрованному каналу HTTPS/TLS. Резервная копия хранится в localStorage вашего браузера и не передаётся на сервер.

⚠ Обращаем внимание: серверы Google Cloud расположены в Европейском союзе. Мы планируем перенос хранилища на российские серверы. До момента переноса использование Сервиса означает согласие с хранением данных за рубежом в соответствии с п. 4 ст. 12 152-ФЗ (трансграничная передача при наличии согласия субъекта).

4. Кому передаются данные

ПолучательСтранаЦель
Google LLC (Firebase, Auth) США / ЕС Аутентификация и хранение данных
DeepSeek (Hangzhou DeepSeek AI) Китай AI-анализ кредитов и голосовой ввод (только при использовании функций)
Cloudflare Inc. США Обработка API-запросов (Worker)

Финансовые данные не передаются в DeepSeek в полном объёме. При AI-анализе передаются только агрегированные числа (суммы долгов, доходов, расходов) без привязки к личности пользователя.

5. Срок хранения данных

Данные хранятся до момента удаления аккаунта пользователем или до истечения 3 лет с момента последнего входа в Сервис. Резервные копии в localStorage автоматически удаляются через 7 дней.

6. Права пользователя

В соответствии с 152-ФЗ вы имеете право:

  • Получить информацию о том, какие ваши данные обрабатываются — по запросу на email
  • Исправить неточные данные — через настройки Google-аккаунта
  • Удалить данные — через раздел Настройки → «Удалить все мои данные»
  • Отозвать согласие — удалив аккаунт в Сервисе
  • Ограничить обработку — не используя AI-функции
  • Обратиться в Роскомнадзор с жалобой если считаете, что ваши права нарушены

7. Безопасность

Для защиты данных применяются:

  • Шифрование передачи данных (HTTPS/TLS 1.3)
  • Аутентификация через Google OAuth 2.0
  • Правила доступа Firestore Security Rules — каждый пользователь видит только свои данные
  • API-ключи хранятся только на стороне сервера (Cloudflare Worker)
  • Content Security Policy ограничивает загрузку сторонних скриптов

8. Cookies и localStorage

Сервис использует localStorage браузера для хранения резервной копии финансовых данных (ключи вида mf_backup_ДАТА). Сторонние трекеры и рекламные cookies не используются. Google Firebase может устанавливать служебные cookies для поддержания сессии аутентификации.

9. Дети

Сервис предназначен для лиц старше 18 лет. Мы не собираем намеренно данные несовершеннолетних. При обнаружении таких данных они будут немедленно удалены.

10. Изменения политики

При существенных изменениях мы уведомим пользователей через интерфейс Сервиса при следующем входе. Дата редакции указана в начале документа. Продолжение использования Сервиса после изменений означает согласие с новой редакцией.
